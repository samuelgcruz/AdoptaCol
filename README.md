# AdoptaCol
AdoptaCol es una plataforma web estática orientada a conectar personas con mascotas rescatadas en Colombia. Permite explorar animales disponibles filtrando por ciudad, edad, tamaño y raza; conocer refugios aliados por región; y contactar veterinarias solidarias. El proyecto está construido con HTML semántico, CSS personalizado y Bootstrap 5, con diseño completamente responsivo para celular, tableta y escritorio.
Funcionalidades: galería de mascotas con filtros en tiempo real · formulario de solicitud de adopción · modal de inicio de sesión · sección de refugios con filtrado directo · directorio de veterinarias con enlace a WhatsApp · menú colapsable en móvil.
Tecnologías: HTML5 · CSS3 · Bootstrap 5.3 · Bootstrap Icons · JavaScript vanilla.
